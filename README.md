# resource-portal-cn

## 项目简介

resource-portal-cn 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为通用资源展示与分发的基础结构存在。

仓库内的每个 HTML 页面均为独立单元，可单独访问，用于承载各类静态信息或轻量交互内容。

## 目录结构

```
resource-portal-cn/
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 公共资源文件（样式、脚本、图片等）
├── index.html      # 仓库首页 / 导航页
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件对应一个独立的 HTML 页面，命名清晰，便于查找。
- **assets/**：存放页面间共享的 CSS、JavaScript 及图片资源，避免重复。
- **index.html**：可选入口页面，用于汇总或导航至各页面。

## 页面归档说明

- 所有 HTML 页面均为静态文件，不依赖后端服务。
- 页面之间无强耦合关系，可独立部署或引用。
- 归档内容遵循通用互联网信息发布规范，不包含任何违规或敏感信息。
- 各页面文件名及路径应尽量反映内容主题，便于维护与索引。

## 维护说明

- 新增页面请放入 `pages/` 目录，并保持命名一致。
- 如需更新公共资源，请修改 `assets/` 下对应文件，注意兼容性。
- 删除或废弃页面时，建议同时清理相关引用与链接。
- 欢迎提交 Pull Request 或 Issue 讨论页面组织与结构优化。

## 其他

本仓库仅用于静态页面归档与展示，不提供动态功能或用户数据收集。如有特殊需求，请参考各页面内部说明。
